# Furniture Ecommerce Project

This project contains both the frontend and backend of the Furniture Ecommerce application.

## Project Structure
- `furniture-frontend/`: React/Vite Frontend Application
- `furniture-backend/`: Node.js/Express Backend API

## How to Run

### 1. Backend Setup
```bash
cd furniture-backend
npm install
# Configure your .env file
npm start
```

### 2. Frontend Setup
```bash
cd furniture-frontend
npm install
npm run dev
